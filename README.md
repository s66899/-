# 🏸 青羽教务系统

羽毛球培训教务管理系统，支持桌面端和 Web 端。

## 功能

- **学员管理**：添加/编辑/删除学员，CSV 导入导出，全局搜索
- **报课管理**：6 种课程包（1v1/1v2/课包/月卡），报课记录
- **排课管理**：按星期/时间段排课，教练分组，CSV 导入
- **消课点名**：记录课时消耗，查看消课历史
- **统计中心**：学员数、课时、消耗率等实时统计

## 快速开始

### Web 端

```bash
pip install flask
python app.py
```

访问 http://localhost:5000

### 桌面端

直接运行 `青羽教务系统.exe`，或 `python main.py`

## 文件结构

```
├── app.py              # Flask Web 后端
├── main.py             # Tkinter 桌面端
├── 青羽教务系统.exe     # 打包好的桌面程序
├── 教务数据.json        # 数据库（共享）
├── requirements.txt    # Python 依赖
├── templates/          # Web 前端模板
└── static/             # Web 静态资源（CSS/JS）
```
